En la tabla anterior, se muestra el % de rendimiento por cada peso que usted asigne a cada Inversión. Por ejemplo:

  • Escenario 1: por cada $10.000 que se asigne a la Inversión 1 usted podrá ganar $1.000 adicionales y por cada $10.000 que se asigne a la Inversión 2 podrá ganar $1.000 adicionales.
  • Escenario 3: por cada $10.000 que se asigne a la Inversión 1 usted podrá ganar $600 adicionales y por cada $10.000 que se asigne a la Inversión 2 podrá ganar $1.000 adicionales.
  • Escenario 5: por cada $10.000 que se asigne a la Inversión 1 usted podrá ganar $600 adicionales y por cada $10.000 que se asigne a la Inversión 2 podrá ganar $400 adicionales.
Cada inversión puede tener un % de rendimiento ALTO y % de rendimiento BAJO y esos porcentajes pueden variar en cada problema.

Para determinar sus ganancias seleccionaremos al azar un problema y uno de los escenarios para ser pagados. Es como si se lanzaran dos dados: UNO para determinar el problema relevante para el pago y UNO para determinar el escenario. Su ganancia adicional entonces dependerá del dinero que usted le asignó a cada inversión en el problema elegido y del escenario elegido.
